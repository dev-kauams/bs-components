O que é o Bootstrap?

Bootstrap é uma biblioteca de código aberto para desenvolvimento de interfaces web responsivas e modernas. Ele fornece um conjunto de ferramentas, estilos e componentes pré-construídos que facilitam a criação de sites e aplicativos web de forma rápida e eficiente.
